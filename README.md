## 什么是浏览器指纹？
网络浏览器在使用时获取的信息包括语言偏好和IP地址。这些信息足以识别唯一的设备。另外设备指纹识别可以在多天内确定设备的属性，以尽可能正确地识别用户信息。  

浏览器指纹识别最常用的数据有两个：存储在用户的设备或浏览器的客户端的cookie，还有一个是存储在服务器端数据库中的设备指纹。  

## 网站收集了什么类型的信息？
每当用户在web浏览器上浏览时，会收集大量用户信息，例如：  
+ 请求标头
+ IP地址
+ 用户代理字符串
+ 已安装的设备插件
+ 用户时区
+ 设备信息，如触摸支持、操作系统、语言和屏幕分辨率，以识别设备类型
+ Flash插件数据
+ 已安装字体
+ 时间标记
